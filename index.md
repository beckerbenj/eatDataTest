# eatDataTest

## Overview

`eatDataTest` (educational assessment tools: Data Tests) is a framework
for testing an comparing different data set versions.

## Installation

``` r

# Install development version from GitHub via
remotes::install_github("beckerbenj/eatDataTest")
```

## Usage

The core functionality of the package currently consists of these two
functions:

``` r

run_data_tests()
create_diff()
```
